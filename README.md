# login

## Descrição:

Neste projeto, foram aplicados conhecimentos de estilização, autenticação e validação simples de formulários, além de boas práticas de UX (User Experience). Foram utilizadas as seguintes tecnologias: `HTML5` `CSS3` `JavaScript (ES6+)`.

## Como usar:

Utilize o usuário `User1` e a senha `12345`. O sistema conta com validação de dados: caso os campos estejam vazios ou as informações sejam incorretas, o usuário será notificado e não haverá redirecionamento.
